# Working on the Backend
## Create the Virtual Environment
```
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

## Running the Virtual Environment
Should be created after making venv. 
```
. venv/bin/activate
```  

## Run the Server
```
python3 app.py
```
